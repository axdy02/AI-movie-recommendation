# AI Movie Recommendation API

Production-style FastAPI backend skeleton for an AI movie recommendation system.

This project intentionally has no frontend and no recommendation logic yet. The only implemented endpoint is the health check.

## Stack

- FastAPI
- SQLAlchemy
- PostgreSQL
- Alembic
- JWT authentication utilities
- passlib/bcrypt password utilities
- Pydantic schemas

## Run Locally

Create and activate a virtual environment, then install dependencies:

```bash
pip install -r requirements.txt
```

Create your environment file:

```bash
cp .env.example .env
```

Start the API:

```bash
uvicorn main:app --reload
```

Health check:

```bash
GET /health
```

Example response:

```json
{
  "status": "ok",
  "service": "AI Movie Recommendation API",
  "environment": "local"
}
```

## Alembic

Create an autogenerated migration after model changes:

```bash
alembic revision --autogenerate -m "describe change"
```

Apply migrations:

```bash
alembic upgrade head
```
